Inline vs Block Elements

Inline elements take up only as much space as their content needs and do not start on a new line. Examples include <span> and <a>. Block elements, on the other hand, start on a new line and take up the full width of their parent container. Examples include <div>, <p>, and <h1>.

  • Inline: Takes up only the space its content needs
  • Block: Takes up the full width of its parent container
